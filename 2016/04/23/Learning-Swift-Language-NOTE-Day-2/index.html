Flow Control

Conditionals

  • if
  • switch

    Loops

  • for-in
  • for
  • while
  • repeat-while

Parentheses around the condition or loop variable are optional. Braces around the body are required.

let iSources = [ 70, 30, 100, 20 ]
var tSource = 0

for score in iSources {
    if score > 50 {
        tSource += 3
    } else {
        tSource += 1
    }
}

print( tSource )

In a if statement, the conditional must be a Boolean expression – this means: That code such as ‘if score { …. }’ is an error, not an implicit comparision to zero.

We can use if and let together to work with values that might be missing. These values are represented as optionals. An optional value either contains a value or contains nil to indicate that a value is missing. Write a question mard( ? ) after the type of a value to mark the value as optional.

var optionalString: String ? = "Hello"
print ( optionalString == nil )

var optionalName: String? = "John"
var greeting = "Hello!"

if let name = optionalName {
    greeting = "Hello, \( name )"
}

Another way to handle optional values is to provide a default value useing the ?? operator. If the optional value is missing, the default value is used instead.

let nickName: String? = nil
let fullName: Stirng  = "John Appleseed"
let informalGreeting = "Hi \( nickName ?? fullName )"
