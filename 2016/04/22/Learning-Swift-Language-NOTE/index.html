Day 1

Simple Values

let: make a constant, It’s doesn’t need to be known at compile time, but must assign it a value exactly once.
var: make a variable.

A constant or variable must have the same type. Default it will infer its type. If the initial value doesn’t provide enough information, We can specify the type by writing it after the variable, separated by a colon.

There’s a even simpler way to include values in string:

Write the value in parentheses, and write a backslash\ before the parentheses. 

let foo = 3
let bar = 5
let foo_bar = "Foo + Bar = \( foo + bar )."

Create array and dictionaries using brackets[], and access their elements by writeing the index or key in brackets. A comma is allowed after the last element.

var vList = [ 'foo', 'bar' ]
or 
var vList = [ 'foo', 'bar', ]

var vDictionary = [
    "foo": 5,
    "bar": 8,
]

vDictionary["foo"]

Create an empty array or dictionary, use the initializer syntax.

let emptyArray = [String]()
let emptyDictionary = [String:Float]()

Also, If the type can be inferred, you can write a empty array as [] and empty dictionary as [:].

var vList = []
var vDictionary = [:]
