How to add a new hard disk on Debian OS

Apr 19, 2016

Adding a additional hard disk to your server is easy and often required.
Here’s steps through the process newly attached drive.

  • Step: 1 Figure out the device name for the new device

    fdisk -l

    root@debian:~# fdisk -l

    Disk /dev/sdb: 1000.2 GB, 1000204886016 bytes
    255 heads, 63 sectors/track, 121601 cylinders, total 1953525168 sectors
    Units = sectors of 1 * 512 = 512 bytes
    Sector size (logical/physical): 512 bytes / 4096 bytes
    I/O size (minimum/optimal): 4096 bytes / 4096 bytes
    Disk identifier: 0x00000000

    Disk /dev/sdb doesn’t contain a valid partition table

    Disk /dev/sda: 128.0 GB, 128035676160 bytes

    255 heads, 63 sectors/track, 15566 cylinders, total 250069680 sectors

    Units = sectors of 1 * 512 = 512 bytes

    Sector size (logical/physical): 512 bytes / 512 bytes

    I/O size (minimum/optimal): 512 bytes / 512 bytes

    Disk identifier: 0x00018dcc

    Device Boot Start End Blocks Id System

    /dev/sda1 * 2048 239855615 119926784 83 Linux

    /dev/sda2 239857662 250068991 5105665 5 Extended

    /dev/sda5 239857664 250068991 5105664 82 Linux swap / Solaris

    NOTE: Disk /dev/XXX this usually is you new HDD Driver

  • Step: 2 Partion the new disk using the cfdisk

    cfdisk /dev/sdb

    New -> Primary -> Specify size in MB -> Write -> yes

    Quit

  • Step: 3 Format the new disk with ext4 filessystem

    mkfs.ext4 /dev/sdb1

  • Step: 4 Create a new directory for mount the new disk in filesystem

    mkdir /new_hdd

  • Step: 5 Findout the UUID (Universally Unique Identifier)

    blkid

    /dev/sda5: UUID=”44e8b854-8fe4-4a36-8d5a-f4cbed9536c6” TYPE=”swap”
    /dev/sdb1: UUID=”8e3c7dd4-a4f8-434c-b74b-7b927dbabe73” TYPE=”ext4”
    /dev/sda1: UUID=”b4ccfaeb-051e-4838-a082-e98fcaec078d” TYPE=”ext4”

  • Step: 6 Edit fstab to automatically mount new HDD on boot

    vim /etc/fstab

    add below line at end of file:

    UUID=8e3c7dd4-a4f8-434c-b74b-7b927dbabe73 /new_hdd ext4 errors=remount-ro 0 1

  • Step: 7 ( Optional) You also can mount by manual:

    mount /dev/sdb1 /new_hdd

Now the new hard disk is mounted and ready to use.
